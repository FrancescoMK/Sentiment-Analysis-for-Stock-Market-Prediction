How it works

Article feed

Embedded API module gathers real-time data from different news station feeds

Model sentiment

Natural language processing model estimates the sentiment of the stocks

Real-time database

Sentiment analysis is stored in structured database, which is updated in real-time

Dashboard display

Results are displayed in a intutive dashboard, which enables insights into the relevant stock sentiment
